Explaining CNNs

A convolutional neural network is a class of artificial neural network that is most commonly used in the context of artificial intelligence as a means to solve problems in image classification.

An artificial neural network is a machine learning model that uses many layers of interconnected neurons to learn information and use this information to make future predictions. A neural network consists of many interconnected layers. Each layer hold a series of neurons. Each neuron is able to perform a certain mathematical computation on its input and is then able to output the resulting data. The structure of these neurons are such that each neuron of a certain layer is fully connected to each neuron of the adjacent layer. The neurons communicate by activating each other with an “activation function”. When one neuron activates an adjacent neuron, its computational output is sent as an input to its adjacent neuron. When such a model is trained on a dataset, it learns which connections between neurons are best at predicting a target. In the case of convolutional neural networks, the target is usually a word that describes the input image. For example, a convolutional neural network that is trained to predict food types will output the word ‘hotdog’ when passed an image of a hotdog.

The main structure of a convolutional neural network can be divided into three main parts. The first part is an input layer. This input layer is a layer that holds as many neurons as there are data points in the input image. As such, a model trained to predict the label of a grayscaled, 10x10 pixels image, will hold 100 neurons. The second part is a network of hidden layers. The layer architecture is completely designed by the programmer of the network and can be optimized for a given problem. The connections between layers are given “weights” (another word for importance) during training. The last part is an output layer. This layer usually holds as many neurons as there are possible labels (hotdog, pizza, sushi, etc. as in the previous example) for the input image.
